#  Iris Flower Classification using Decision Tree

> Your first step into machine learning — clean, classic, and built right.

---

## 🚀 Project Overview

This project is a beginner friendly machine learning classification task using the famous Iris dataset.

Predict the species of a flower based on its physical measurements.

This project follows a **complete end-to-end machine learning workflow**, making it a strong foundation for future ML projects.

---

## 📊 Dataset Description

The Iris dataset is one of the most well-known datasets in machine learning.

### 🔹 Features

* Sepal Length (cm)
* Sepal Width (cm)
* Petal Length (cm)
* Petal Width (cm)

### 🎯 Target Classes

* 🌿 Setosa
* 🌸 Versicolor
* 🌺 Virginica

---

## 🧠 Machine Learning Model

This project uses a:

### 👉 Decision Tree Classifier

A Decision Tree works by splitting the data into smaller groups based on feature values, forming a tree-like structure of decisions.

✔ Easy to understand
✔ Interpretable
✔ Perfect for beginners

## 🔄 Project Workflow

```text
Data → Exploration → Preprocessing → Model Training → Evaluation → Visualization
```

### Step-by-step:

1. Import libraries
2. Load dataset
3. Convert to DataFrame
4. Explore data (EDA)
5. Define features (X) and target (y)
6. Train-test split
7. Train Decision Tree model
8. Make predictions
9. Evaluate performance
10. Visualize decision tree and feature importance
